# GEMA Club Student Portfolio

A professional student portfolio website developed for the GEMA Club Assignment.

## 🌐 Live Website

https://gema-club-portfolio.onrender.com

## 📋 Features

- Homepage
- About Me
- Academic Performance
- GEMA Club Participation
- Competition Achievements
- Project Showcase
- Certificates Gallery
- Skills Section
- Student Blog
- Testimonials
- Contact Form
- Downloadable Resume

## 🚀 Technologies Used

- HTML5
- CSS3
- JavaScript
- GitHub
- Render

## 💻 Local Development

Run a local server:

```bash
python -m http.server 8000
```

Then open:

http://localhost:8000
